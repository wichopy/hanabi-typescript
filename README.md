# Hanabi

Implementation of the board game [Hanabi](https://en.wikipedia.org/wiki/Hanabi_(card_game)) using Typescript

## Setup

```
git clone [this repo]
cd [this repo]
npm i
```

## Testing
To run tests and watch for file changes:

`npm run test:watch`
